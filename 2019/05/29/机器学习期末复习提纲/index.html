第一章

  • 机器学习的概念

机器学习是这样一门学科，它致力于研究如何通过计算的手段，利用经验来改善系统自身的性能。在计算机系统中，“经验”通常以“数据”形式存在。

  • 机器学习的主要研究内容

是关于在计算机上从数据中产生“模型”的算法，即“学习算法”。

  • 什么叫做学习（P2）

从数据中学得模型的过程称为“学习”(learning)或“训练”(training)，这个过程通过执行某个学习算法来完成。训练过程中使用的数据称为“训练数据”(training data)，其中每个样本称为一个“训练样本”(training sample)，训练样本组成的集合称为”训练集”(training set)。

学得模型后，使用其进行预测的过程称为“测试”(testing)，被预测的样本称为”测试样本”(testing sample)。

  • 监督学习和非监督学习(P3)

根据训练数据是否拥有标记信息，学习任务可大致划分为两大类：“监督学习”和“无监督学习”，分类和回归是前者的代表，而聚类则是后者的代表。除此之外，还有强化学习、半监督学习。

  • 常见的监督学习算法
    • Linear Regression
    • Logistic Regression
    • Support Vector Machines (SVMs)
  • 常见的无监督学习算法
    • k-Means
  • 回归和分类的区别(P3)
    • 若预测的是离散值，例如“好瓜”“坏瓜”，此类学习任务称为“分类”(classification)；若欲预测的是连续值，例如西瓜成熟度0.95、0.37，此类学习任务称为“回归”(regression)。

判定西瓜是否为好瓜：分类问题；拟合二次曲线：回归问题；让机器下棋（阿尔法狗）：强化学习；

  • 独立同分布（P3）

通常假设样本空间中全体样本服从一个未知“分布”(distribution)D，我们获得的每个样本都是独立地从这个分布上采样获得的，即”独立同分布“(independent and identically distributed)。

第二章 模型评估与选择

  • 误差、精度、错误率、它俩之间的关系
    • 错误率：分类错误的样本数占样本总数的比例
    • 精度：分类正确的样本数占样本总数的比例
    • 精度=1-错误率
    • 误差：学习器的实际预测输出与样本的真实输出之间的差异成为“误差”(error)。
  • 经验误差、泛化误差、训练误差、测试误差、他们之间的关系、测试误差和泛化误差的区别
    • 测试误差是在测试集上的误差，泛化误差不一定是测试集，由于我们不能测试尽所有的样本，就用测试误差用来近似泛化误差
    • 学习器在训练集上的误差称为“训练误差”(training error)或“经验误差”(empirical error)，在新样本上的误差称为“泛化误差”(generalization error)。
  • 模型评估的时候用来测试模型的数据集叫做什么集？评估这个模型的性能组成的集合？B A训练集、B测试集、C评估集、D验证集
  • 留出法、分层采样（P25)
    • “留出法”直接将数据集D划分为两个互斥的集合，其中一个集合作为训练集S，另一个作为测试集T。
    • 训练/测试集的划分要尽可能保持数据分布的一致性。例如通过对D进行分层采样而获得含70％样本的训练集S和含30％样本的测试集T，若D包含500个正例、500个反例，则分层采样得到的D应包含350个正例、350个反例，而T则包含150个正例、150个反例。
  • 交叉验证法 假设有一个数据集10类，用5折（或其他折）交叉验证评估模型，应该怎么分数据集？谁当训练集？谁当测试集？（大题）
  • 将这个数据集划分为5份，每份中包含各类的1/5的数据，每次用其中4份做训练集，剩余的1份做测试集，使每一份都做一次测试集，则一共5次训练/测试，最终结果取5次测试结果的均值
  • 将数据集D划分为k个大小相似的互斥子集。每次用k-1个子集的并集作为训练集，余下的那个子集作为测试集，最终返回的是这k个测试结果的均值。k折交叉验证
  • 留一法
    • 假定数据集D中包含m个样本，若令k=m，则得到了交叉验证法的一个特例：留一法(Leave-One-Out)。也即m个样本划分为m个子集，只有一种划分方法。
  • 自助法
    • 每次采样有一些样本永远不会采到：公式+数字

第四章 决策树

  • 信息熵（P75） 熵的概念

    • 当一件事情有多种情况时，这件事情对某人而言具体是哪种情况的不确定性叫做熵

    • 熵和信息数量相等，意义相反，获取信息意味着消除不确定性（熵）

    • 信息熵用来衡量信息量的大小
      若不确定性越大，则信息量越大，熵越大
      若不确定性越小，则信息量越小，熵越小
      比如A班对B班，胜率一个为x，另一个为1-x
      则信息熵为 -(xlogx + (1-x)log(1-x))
      求导后容易证明x=1/2时取得最大，最大值为2

      也就是说两者势均力敌时，不确定性最大，熵最大。
